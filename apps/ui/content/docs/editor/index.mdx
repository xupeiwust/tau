# Tau Editor

The Tau Editor is a web-based CAD modeling environment with AI-assisted design, real-time 3D preview, and multi-kernel support.

Editor documentation is coming soon. In the meantime, explore the [Runtime documentation](/docs/runtime) to learn about the runtime that powers the editor's CAD capabilities.

- **Parametric Design** -- Working with parameters and constraints
