# Kernel Factories

Use kernel factories when registering CAD engines with `createKernelClient`. Each factory returns a `KernelPlugin` that the client uses for kernel selection and worker loading.

## Factories

| Factory | Extensions | Purpose |
|---------|------------|---------|
| `replicad(options?)` | `ts`, `js` | OpenCASCADE-based parametric BRep |
| `openscad()` | `scad` | CSG modeling |
| `jscad()` | `ts`, `js` | @jscad/modeling parametric CSG |
| `zoo(options)` | `kcl` | KCL via Zoo API |
| `tau()` | `*` | Catch-all for STEP, STL, 3MF |

## Usage

```typescript
import { createKernelClient, fromMemoryFS } from '@taucad/kernels';
import { replicad, openscad, jscad, zoo, tau } from '@taucad/kernels/kernels';

const client = createKernelClient({
  kernels: [
    replicad({ withExceptions: true }),
    openscad(),
    jscad(),
    zoo({ baseUrl: 'wss://example.com/v1/kernels/zoo' }),
    tau(),
  ],
  bundlers: [],
  fileSystem: fromMemoryFS(),
});
```
