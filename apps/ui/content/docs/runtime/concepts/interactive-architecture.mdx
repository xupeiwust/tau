## How It Works

This interactive diagram is built with [React Flow](https://reactflow.dev/) and renders as a custom MDX component (`<InteractiveDiagram>`). Compared to the static Mermaid version on the [Architecture](./architecture) page:

- **Click** any node to highlight its direct connections and dim unrelated parts
- **Hover** to see a description tooltip explaining each component's role
- **Selected node details** appear in a panel below the diagram
- **Dark mode** follows the site theme automatically

The diagram data is defined inline as JavaScript objects — nodes with positions, labels, and descriptions, and edges with labels and connection types. This makes it easy to maintain alongside the documentation text.
