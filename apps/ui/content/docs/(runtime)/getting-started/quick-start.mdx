# Quick Start

Render a 3D model from TypeScript code using the Replicad kernel. You will create a runtime client, write a simple CAD model, and render it to glTF.

## Prerequisites

- [Install @taucad/runtime](/docs/getting-started/installation)

## Render a 3D Model

Create a file called `render.ts` and paste the following code:

```typescript
import { createRuntimeClient, fromMemoryFS } from '@taucad/runtime';
import { replicad } from '@taucad/runtime/kernels';
import { esbuild } from '@taucad/runtime/bundler';

const client = createRuntimeClient({
  kernels: [replicad()],
  bundlers: [esbuild()],
  fileSystem: fromMemoryFS(),
});

client.on('log', ({ message }) => console.log(message));

const result = await client.render({
  code: {
    'main.ts': `
    import { drawRoundedRectangle } from 'replicad';

    export default function main() {
      return drawRoundedRectangle(30, 50, 5).sketchOnPlane('XY').extrude(10);
    }
  `,
  },
});

if (result.success) {
  console.log(`Rendered ${result.data.length} geometry`);
  console.log(`Format: ${result.data[0].type}`);
} else {
  console.error('Render failed:', result.issues);
}

client.terminate();
```

## Expected Output

```
Rendered 1 geometry
Format: model/gltf-binary
```

Each entry in `result.data` is a `Geometry` object containing a glTF binary (GLB) that you can display in any 3D viewer, save to disk, or pass to a renderer like Three.js.
